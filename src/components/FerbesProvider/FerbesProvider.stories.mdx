# LayoutProvider

### Setting a theme

`Layout Components` comes with a pre-configured theme. Although in most cases
you want to provide your own theme:

At the root of your application, wrap the components in the `LayoutProvider`.

```jsx
import { LayoutProvider } from '@3pc/layout-components-react';

const theme = {
  breakpoints: ['450px', '900px', '1200px'],
  space: [2, 4, 6, 8, 12, 16, 24, 32],
};

const App = () => <LayoutProvider theme={theme}>...</LayoutProvider>;
```

> Your custom theme will be merged with the default theme.

### Adding global styles

```jsx
import { LayoutProvider, css } from '@3pc/layout-components-react';

const styles = css`
  body {
    background: black;
  }
`;

const App = () => <LayoutProvider styles={styles}>...</LayoutProvider>;
```

### Style with style props

`Layout Components` allows you to control styles based on a global theme.
Some components have style props made available by [Styled System](https://styled-system.com/table):

```javascript
<Box color="#fff" bg="tomato">
  Tomato
</Box>
```

So far, this component can be styled with any valid CSS color. To create a more consistent UI, use the style tokens provided by your theme.

You can extend your theme like this:

```javascript
const theme = {
  colors: {
    primary: ['#5b53ff', '#2745fa', '#5378fc', '#93b4fb', '#e6effe'],
    white: '#ffffff',
    ...
  }
  ...
}
```

Then you can use it like this:

```javascript
<Box color="white" backgroundColor="primary.1">
  Blue Box
</Box>
```

### Hide Focus Rings

`Layout Components` comes with support for hiding focus rings, when using the
app with your mouse. You can enable this feature by providing a `hideFocusRings`
prop:

```jsx
import { LayoutProvider } from '@3pc/layout-components-react';

const App = () => <LayoutProvider hideFocusRings={true}>...</LayoutProvider>;
```
