# Bleed

Applies a negative margin to allow the content to bleed out into a the surrounding layout. As such it is important to ensure that size and direction of the bleed is accommodated by the parent layout.

<Canvas withSource="open">
  <Story name="Base">
    <Box px="8">
      <Stack space="4">
        <Placeholder height={40} />
        <Bleed horizontal={8}>
          <Placeholder height={40} />
        </Bleed>
        <Placeholder height={40} />
      </Stack>
    </Box>
  </Story>
</Canvas>

### Vertical

Contents will bleed into the surrounding layout vertically using the <b>vertical</b> prop.

<Canvas>
  <Story name="Vertical">
    <Box
      padding="8"
      css={{
        bg: 'dodgerblue',
      }}
    >
      <Bleed vertical="8">
        <Placeholder height={100} />
      </Bleed>
    </Box>
  </Story>
</Canvas>

### Specific Direction

Contents will bleed into the surrounding layout in a specified direction using the <b>top</b>, <b>bottom</b>, <b>left</b>, or <b>right</b> prop.
