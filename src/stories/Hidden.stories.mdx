# Hidden

Allows you to hide content responsively.<br />
The number you pass to `above` and `below` corresponds to the index of the breakpoints array of your theme.

<Canvas>
  <Story name="Below">
    <Hidden below={1}>
      <p>{`Hidden below 768`}</p>
    </Hidden>
  </Story>
</Canvas>

<Canvas>
  <Story name="Above">
    <Hidden above={2}>
      <p>{`Hidden above 1280`}</p>
    </Hidden>
  </Story>
</Canvas>

<Canvas>
  <Story name="Below and Above">
    <Hidden below={1} above={2}>
      <p>{`Hidden below 768 and above 1280`}</p>
    </Hidden>
  </Story>
</Canvas>
