# useTheme

<br />
<br />

If you need direct access to the theme tokens, you can use the `useTheme` hook.

```jsx
import { useTheme, Box } from 'ferbes-layout';

const MyComponent = () => {
  const { colors } = useTheme();

  return <Box backgroundColor={colors.primary[1]} />;
};
```

> Nevertheless, to ensure consistency in your app, it is recommended to only use the available theme tokens.
