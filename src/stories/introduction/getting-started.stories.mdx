# Getting Started

The instructions below cover how to use `Ferbes Layout` within a React application.

### Prerequisites

- You should have **[Node.js](https://nodejs.org/) v10.17.0+** installed
- To add this package to your project you need either **[npm](https://www.npmjs.com/)** or **[yarn](https://yarnpkg.com/)**.<br />
  `npm` comes with node. You can read how to install `yarn` in [their documentation](https://yarnpkg.com/en/docs/install).

### Adding `Ferbes Layout` to an existing app

```bash
npm i ferbes-layout
# Or if you prefer yarn:
yarn add ferbes-layout
```

### Configuring the theme

Refer to the [FerbesProvider documentation](/?path=/docs/logic-ferbesprovider--page) to learn more about configuring `Ferbes Layout`.
